# План изучения React (после освоения основ)

## Проект для закрепления базы – «Умный список дел» (Smart Todo)

### Основные темы, которые используются в проекте
- JSX
- Props и children
- Вывод списков (map + key)
- Обработка событий
- Хуки: useState, useEffect, useLayoutEffect, useRef
- Подъём состояния
- Управляемые формы

### Техническое задание (кратко)
- Добавление задачи (заголовок, описание, приоритет)
- Удаление / редактирование
- Отметка о выполнении
- Фильтрация (все / активные / выполненные)
- Поиск по заголовку
- Счётчик невыполненных задач
- Кнопка «Очистить выполненные»
- Сохранение в localStorage
- Демонстрация useLayoutEffect (измерение высоты списка)
- Компонент-обёртка с children
